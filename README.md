# Trip-Advisor-Clone
A responsive travel discovery website inspired by TripAdvisor, featuring destinations, hotels, restaurants, search filters, and a modern, mobile-friendly design built with HTML and CSS.

Wanderlust - Travel Discovery Website
Project Overview

Wanderlust is a responsive travel and tourism website inspired by TripAdvisor, designed to help users discover popular destinations, hotels, and restaurants through an attractive and user-friendly interface. The project showcases modern front-end development practices with a focus on accessibility, responsiveness, and engaging visual design.

Key Features
🌍 Featured travel destinations section
🏨 Popular hotels showcase with ratings and pricing
🍽️ Top-rated restaurants listing
🔍 Destination search interface
🎯 Advanced filtering by category, rating, and price
📱 Fully responsive design for desktop, tablet, and mobile devices
♿ Accessibility-friendly navigation and focus states
✨ Interactive cards with hover animations
🎨 Modern UI using CSS Grid and Flexbox layouts
📊 Rating and review display system
